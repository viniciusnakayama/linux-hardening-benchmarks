# Linux Security Hardening

Repository with self-made automations, in Bash, to reinforce the security 
of my Linux environment (Ubuntu) — made in my free time as a way to 
practically apply security concepts I have been studying.

## Motivation

Each automation here solves a real security problem I identified on my own 
system, following the CIS Benchmark (consensus-driven security guidelines 
published by the Center for Internet Security). The goal is not just to 
run the script, but to understand what each tool does internally and 
document my reasoning.

## Environment

Each automation is isolated in its own folder, containing:

- `scripts/` — the executable `.sh` files, commented
- `docs/` — step-by-step guide with screenshots and explained commands
- `README.md` — overview of that specific project
